# Personal Expense Tracker

Personal Expense Tracker is a web application built using Next.js and React. It helps users add, manage, categorize, and delete daily expenses. The system automatically calculates total expenses and organizes records by category, providing a simple, responsive, and user-friendly expense management solution.

## Features

- Add new expenses
- Delete expenses
- Category-wise expense management
- Automatic total expense calculation
- Responsive user interface
- Real-time expense updates

## Categories

- Food
- Grocery
- Petrol/Diesel
- Mobile Recharge
- WiFi Recharge
- Medical
- Travel
- Shopping
- Bills

## Technologies Used

- Next.js
- React
- TypeScript
- CSS
- Git & GitHub
